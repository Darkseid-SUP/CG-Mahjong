  The Goal

The aim of this game is to win.

For this question, you will be asked to output a single line as a solution.

  Rules

  • A very helpful youtube video
  • A very helpful article

tiles is represented by 2 characters, first being the representation of the tile type, while the second being the corresponding value


List of tile types, and their character representation

  • Dot
  • Bamboo
  • Character
  • Wind
  • dRagon
  • Flower
  • Season

For the suited tiles, the value will be the numeral value. For example, "5 Dot" is represented as d5, while "1 Bamboo is represented as b1"


for honour tiles and season tiles, the value represents the index of the tile type, the types are ordered as follow:

  • Wind tile: East, South, West, North
  • Dragon tile: Red dragon, Green dragon, White dragon
  • Flower tile: 梅,兰,竹,菊
  • Season tile: 春,夏,秋,冬

For example, South will be represented as w2, and White dragon will be represented as r3


Each round, you will receive a list of actions, the actions given is the action made by other players since your last turn, with the actions you committed being on top of the list. You can only get non-draw actions by other players, as it you aren't suppose to know what they drew. Refer below for the action format


there are 2 types of roundType, normal and interrupt


In normal round, the last actions of the list will be about the tile you drew. in normal mode, you can only perform a DISCARD action


In interrupt round, the last action of the list will be a discard. It means that you have an oppurtunity to interrupt the normal move sequence, with actions like WIN, GONG, PONG and SEUNG


Since this is WIP, you still can't do a win/gong action


If you don't want to perform any action, you can output PASS (only available in interrupt round). Even if you output your action, you won't always be able to successfully perform it, as other's might have a higher priority than you. To know if you performed it, you can refer to actions of the next normal round

  Game Input

Initialization input
Line 1 : windSeat, aka your player id.
Line 2 : startingTiles, your starting hand. (representation of tiles chucked together with no spaces, for example: f3d1d8d8b3b6b7c1c6c8r1r1r2)
Input for one game turn

Line 1: roundType, normal | interrupt

Line 2: integer numberOfActions

Next numberOfActions lines:

  • integer committer, representing the id of the player that did this action
  • string action, can be either:
    • Draw
    • Discard
    • Pong
    • Seung
    • Gong
    • Flower
Output
A single line containing the player action. List of actions:

Normal round:

  • DISCARD tile

Interruption round:

  • PONG tile, tile being the tile you wish to discard
  • SEUNG tile1tile2tile3, tile1 and tile2 being the tile you wish to seung with the target tile, and tile3 being the tile you wish to discard
  • GONG, not available yet
  • WIN, not available yet
Constraints
windSeat 0~3
